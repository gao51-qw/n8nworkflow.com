# Identify and notify WooCommerce VIP customers with Airtable and Slack

> # WooCommerce VIP Customer Automation

This workflow automatically identifies VIP customers from your WooCommerce store based on their total spending and number of completed orders. It pulls new orders on a schedule, filters valid transactions, groups customers, calculates their VIP tier, stores them into Airtable and notifies your team in Slack.It removes all manual checking and ensures your team instantly knows when a high-value customer places an order.

## What It Does

This workflow fully automates VIP customer detection for WooCommerce stores. Every few minutes, it retrieves new orders via WooCommerce API. It filters only processing or completed orders and extracts essential fields such as customer ID, customer name order total and timestamps.

The workflow then groups orders by customer to ensure duplicate orders don‚Äôt inflate metrics. For each customer, it fetches their complete order history, calculates lifetime value, counts total paid orders and assigns a VIP tier (Platinum, Gold or Silver).

Once a customer qualifies, their details are saved into Airtable for tracking. A Slack message is simultaneously sent to inform your team so they can take immediate action ‚Äî such as providing priority support, follow-up messages or special offers.

This system runs completely on its own and keeps customer insights up-to-date without manual checks.

## How It Works

1.  **Scheduled Trigger** ‚Äì Runs every few minutes to check for new orders.
2.  **Domain Setup** ‚Äì Assigns the WooCommerce store domain used across API calls.
3.  **Fetch Orders** ‚Äì Retrieves all orders from WooCommerce.
4.  **Filter Valid Orders** ‚Äì Only keeps completed or processing orders.
5.  **Format & Clean Data** ‚Äì Extracts only key order details.
6.  **Deduplicate Customers** ‚Äì Only the first order per customer is processed.
7.  **Fetch Order History** ‚Äì Gets lifetime order history for each customer.
8.  **Calculate VIP Tier** ‚Äì Determines Platinum/Gold/Silver based on spend or order count.
9.  **Filter VIP Only** ‚Äì Removes customers who do not qualify.
10.  **Save to Airtable** ‚Äì Creates/updates VIP records.
11.  **Notify Team on Slack** ‚Äì Sends VIP alerts instantly.
    

## Setup Steps

1.  Import this workflow JSON into n8n.

3.  Update the **wc\_domain** value in the ‚ÄúSet WooCommerce Domain‚Äù node.

5.  Set your Slack channel ID inside the Slack node.
6.  Adjust VIP rules inside the ‚ÄúCalculate VIP Tier‚Äù code node if needed.
7.  Activate the workflow ‚Äî it will now run automatically.
