# Telegram sticker bot

> ![workflow-screenshot](fileId:17)

1. Gets triggered by Telegram once a user sends a message
1. Checks if the message contains a sticker
1. If a sticker got found it sends a message with its ID else it informs the user that no sticker was found in message
