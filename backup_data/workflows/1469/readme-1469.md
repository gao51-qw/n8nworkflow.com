# Post RSS feed items from yesterday to Slack

> This workflow will collect the RSS feed data from the previous day and post them to a Slack channel.

To use this workflow you will need to add your credentials to the Slack node and select the channel for notifications. You will also need to input the URL for the RSS feed.
