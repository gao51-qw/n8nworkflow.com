# Send automated daily reminders on Telegram

> A goal for 2022 is to write 1 thing I do each day. This workflow will automatically remind you on telegram to write something you did yesterday, optionally you can enable the [second workflow](https://n8n.io/workflows/1388) which will allow you to reply to the message and have it recorded in a google sheet.

Note: Make sure to configure your Telegram credentials!
